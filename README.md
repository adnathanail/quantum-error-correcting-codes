# Quantum error correcting codes

Install dependencies (including dev)
```shell
uv sync
```

## prek/pre-commit setup (recommended)

[Install prek](https://prek.j178.dev/installation/)

Set up pre-commit hook
```shell
prek install
```

## Manual commands
Run tests
```shell
uv run pytest
```

Run type-checking
```shell
uv run ty check
```

Run linting
```shell
uv run ruff check
```

Run auto-formatting
```shell
uv run ruff format
```
